# ¿Que es OSINT?

**OSINT** (Open Source Intelligence) es la recolección y análisis de información obtenida de fuentes públicas o abiertas para un propósito específico
- Redes sociales
- Registros públicos
- Meta datos
- Foros
- Documentos públicos

# Herramientas OSINT

## <font color="#00b050">Google Dorks</font>
Es una técnica de búsqueda avanzada que utiliza operadores del motor de Google para encontrar información difícil de localizar con una búsqueda normal.

Buscar Archivos de un formato dentro de un sitio:
```
filetype:pdf site:edu.co"calendario academico"
```

Buscar archivos .txt que contengan una palabra:
```
intext:"password" filetype:txt
```

Buscar archivos que contengan una palabra en su titulo:
```
intitle:"index of"backup"
```

Buscar url con una dirección especifica:
```
inurl:"/view/index.shtml"
```

Algunas extra
```
# Buscar una frase exacta ""
"password"

# Excluir término de la busqueda -
joystic -controller

# Actua como comodin *
email * @company.com
```

## <font color="#953734">TheHarvester</font>
Es una herramienta desarrollada en python diseñada para recolectar direcciones de correo electrónico, subdominios, direcciones IP, nombres de host y URLs.

### Parámetros principales y su explicación

| Parámetro | Descripción                         | Ejemplo              |
| --------- | ----------------------------------- | -------------------- |
| `-d`      | Dominio objetivo (requerido)        | `-d google.com`      |
| `-l`      | Límite de resultados a buscar       | `-l 500`             |
| `-b`      | Fuente de datos a utilizar          | `-b google`          |
| `-s`      | Realizar un escaneo de DNS          | `-s`                 |
| `-v`      | Verificar el nombre de host via DNS | `-v`                 |
| `-f`      | Guardar resultados en un archivo    | `-f resultados.html` |
| `-g`      | Utilizar Google Dorks               | `-g`                 |
**Ejemplo de uso sobre su dominio**
```
theHarvester -d spotify.com -b duckduckgo -l 50
```

Escaneo para descubrir subdominios y puntos de entrada guardado en documentos
```
theHarvester -d microsoft.com -b dnsdumpster,certspotter,crtsh -l 300 -f microsoft_subs.html
```

- dnsdumpster: Herramienta que hace mapeo de dominios
- crtsh: Buscador de certificados, interfaz directa a CT Logs(Certificate Transparency) o registro publico de certificados
- certspotter: Servicio de monitoreo de CT Logs
