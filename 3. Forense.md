# ¿Qué es Forense?
La ciencia forense es el arte de recuperar el rastro digital dejado en una computadora. Existen muchos métodos para encontrar datos que aparentemente se eliminan, no se almacenan o, peor aún, se registran de forma encubierta.
Una parte importante de la ciencia forense es tener las herramientas adecuadas, además de estar familiarizado con su uso. Aborda los desafíos forenses con una mente abierta. ¡No es raro que los desafíos CTF oculten banderas en los rincones más oscuros!
# Aplicaciones en el mundo real
A diferencia de los CTF que normalmente los retratan, los análisis forenses del mundo real rara vez son esotéricos. Por ejemplo, es posible que tengas que volver a ensamblar las particiones de arranque de un disco duro para recuperar sus datos y su sistema de archivos. Por lo tanto, los análisis forenses de CTF normalmente son problemas de acertijos y "retos mentales" que tienen como objetivo introducir una herramienta o método.
# Formatos de archivos
Las extensiones de archivo no son la única forma de identificar el tipo de un archivo; los archivos tienen ciertos bytes iniciales llamados firmas de archivo que permiten a los programas analizar los datos de manera consistente. Los archivos también pueden contener datos "ocultos" adicionales llamados "metadatos" que pueden resultar útiles para encontrar información sobre el contexto de los datos de un archivo.
# Firmas de archivos (File Signatures)
Las firmas de archivos (también conocidas como números mágicos de archivos) son bytes dentro de un archivo que se utilizan para identificar el formato del archivo. Generalmente tienen entre 2 y 4 bytes de longitud y se encuentran al principio de un archivo.
## ¿Para qué se utiliza?
A veces los archivos pueden venir sin extensión o con extensiones incorrectas. Utilizamos el análisis de firmas de archivos para identificar el formato (tipo de archivo) del archivo. Los programas necesitan saber el tipo de archivo para poder abrirlo correctamente. Es útil analizar el tipo de archivo antes que cualquier software forense.
### ¿Cómo obtengo las firmas de archivos?
Existe una forma de poder ver las firmas, aquí se encuentran algunas muy practicas:
* #### comando xxd 
  generará un volcado hexadecimal del archivo el truco está en ver el encabezado de este volcado hexadecimal, para ello se tiene el siguiente conjunto de comandos:
	```
	xxd archivoAVerificar.algo | head
	```

# Metadatos
Los metadatos son datos sobre datos. Los diferentes tipos de archivos tienen diferentes metadatos. Los metadatos de una foto pueden incluir fechas, información de la cámara, ubicación GPS, comentarios, etc. Para la música, pueden incluir el título, el autor, el número de pista y el álbum. Los desafíos de CTF a menudo te hacen buscar pistas específicas en los metadatos de un archivo (especialmente archivos multimedia).

# Timestamps
Los timestamps son datos que indican la hora de ciertos eventos (MAC): 
- Modification: cuando se modificó un archivo 
- Access: cuando se leyó o se accedió a un archivo o entradas 
- Creation: cuando se crearon archivos o entradas
## Tipos de timestamps:
- Modified
- Accessed
- Created
- Date Changed (MFT)
- Filename Date Created (MFT)
- Filename Date Modified (MFT)
- Filename Date Accessed (MFT)
- INDX Entry Date Created
- INDX Entry Date Modified
- INDX Entry Date Accessed
- INDX Entry Date Changed
## ¿Por qué me debe importar esto?
Ciertos eventos como crear, mover, copiar, abrir, editar, etc. pueden afectar los tiempos de MAC. Si se pueden alcanzar las marcas de tiempo MAC, se podría crear una línea de tiempo de eventos.
