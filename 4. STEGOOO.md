# ¿Cómo puedo ocultar mis propios archivos?
existen herramientas como steghide el cual es una herramienta especializada en estenografía que permite ocultar archivos con contraseña
1. Instalarlo:
```
sudo apt install steghide
```
1. Insertar el archivo secreto:
```
steghide embed -cf archivo.contenendor -ef archivo.secreto
```
2. Extraer el archivo:
```
steghide extract -sf archivo.contenedor
```
